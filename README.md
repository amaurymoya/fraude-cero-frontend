# Fraude Cero – Frontend (Next.js 14 + Tailwind)

**Repo listo para Render** (package.json en la raíz).

## Render
- Root Directory: *(dejar en blanco)*
- Build: `npm install && npm run build`
- Start: `npm start`
- Env: `API_ENDPOINT=https://fraudecero.onrender.com/api/analizar` (y opcional `NODE_VERSION=20`)

## Local
```bash
npm install
npm run dev   # http://localhost:5173
```
